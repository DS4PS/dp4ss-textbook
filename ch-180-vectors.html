1.1 Logical Variables

Class logical variables are comprised of one of two values: TRUE or FALSE.


We can create logical variables manually. Determine their class with function class():

x <- TRUE
y <- FALSE

class(x)
## [1] "logical"
class(y)
## [1] "logical"


Often, logical values simply indicate “yes” or “no”

character <- c("Raskolnikov", "Marmeladov", "Razumikhin")
criminal <- c(TRUE, FALSE, FALSE)

data.frame(character,
           criminal)        # Create data frame of characters and crime status
##     character criminal
## 1 Raskolnikov     TRUE
## 2  Marmeladov    FALSE
## 3  Razumikhin    FALSE
